Core Library API

The @resonance/core package provides cryptographic identity, differential privacy perturbation, semantic embedding, vector operations, and the wire protocol. Everything a personal node needs.

Crypto

Cryptographic primitives for identity, signing, and encryption. Built on tweetnacl.

perturbation.ts

Perturbation

Differential privacy via calibrated Laplace noise. This module adds noise to embedding vectors before they leave the personal node.

// Perturb a vector with Laplace noise at a given epsilon function perturbVector( vector: Float32Array, epsilon: number ): PerturbationResult interface PerturbationResult { perturbed: Float32Array; // The noisy vector scale: number; // Laplace scale parameter used epsilon: number; // Privacy parameter } // Perturb using named privacy levels function perturbWithLevel( vector: Float32Array, level: "low" | "medium" | "high" ): PerturbationResult // Calculate the Laplace scale parameter for given epsilon and dimensions function calibrateScale( epsilon: number, dimensions?: number // default: 768 ): number // Sample from a Laplace distribution function sampleLaplace(scale: number): number
Privacy Levels

The named levels map to epsilon values: low = 5.0 (sim ~0.994, minimal privacy), medium = 1.0 (sim ~0.880, recommended), high = 0.1 (sim ~0.179, strong privacy but low utility). Medium is the default for most use cases.

Embedding

vectors.ts

Vectors

Vector math utilities optimized for Float32Array. Used internally by embedding, perturbation, and matching.

// Cosine similarity between two vectors (-1 to 1) function cosineSimilarity( a: Float32Array, b: Float32Array ): number // Dot product function dotProduct( a: Float32Array, b: Float32Array ): number // L2 (Euclidean) norm function l2Norm(v: Float32Array): number // Normalize a vector to unit length function normalize(v: Float32Array): Float32Array
Performance

All vector functions operate on Float32Array for memory efficiency and cache-friendly access. A 768-dim vector takes 3KB as Float32Array vs 6KB as a regular number array.
