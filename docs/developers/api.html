@resonance/core

API Reference

Complete API documentation for the @resonance/core package—cryptographic identity, differential privacy perturbation, semantic embedding, vector operations, and the wire protocol.

crypto.ts

Crypto

Cryptographic primitives for identity, signing, and encryption. Built on tweetnacl (zero dependencies, audited).

generateIdentity()

Generate a new Ed25519 identity with keypair and DID.

function generateIdentity(): Identity

sign()

Sign a message with an Ed25519 secret key.

function sign( message: Uint8Array, secretKey: Uint8Array ): Uint8Array

verify()

Verify an Ed25519 signature against a public key.

function verify( message: Uint8Array, signature: Uint8Array, publicKey: Uint8Array ): boolean

boxEncrypt() / boxDecrypt()

Asymmetric encryption using NaCl box (X25519 + XSalsa20-Poly1305).

function boxEncrypt( message: Uint8Array, recipientPublicKey: Uint8Array, senderSecretKey: Uint8Array ): Uint8Array function boxDecrypt( encrypted: Uint8Array, senderPublicKey: Uint8Array, recipientSecretKey: Uint8Array ): Uint8Array | null

generateEphemeralKeyPair()

Generate an ephemeral X25519 keypair for Diffie-Hellman key exchange in direct channels.

function generateEphemeralKeyPair(): { publicKey: Uint8Array; secretKey: Uint8Array; }

deriveSharedSecret()

Derive a shared secret from a Diffie-Hellman key exchange.

function deriveSharedSecret( mySecretKey: Uint8Array, theirPublicKey: Uint8Array ): Uint8Array

secretboxEncrypt() / secretboxDecrypt()

Symmetric encryption using NaCl secretbox (XSalsa20-Poly1305).

function secretboxEncrypt( message: Uint8Array, key: Uint8Array ): Uint8Array function secretboxDecrypt( encrypted: Uint8Array, key: Uint8Array ): Uint8Array | null

exportIdentity() / importIdentity()

Export an identity encrypted with a password (for backup), and import it back.

function exportIdentity( identity: Identity, password: string ): string function importIdentity( encrypted: string, password: string ): Identity
embedding.ts

Embedding

Semantic embedding using @huggingface/transformers. Runs locally on the personal node—no API calls, no data leaves the device.

EmbeddingEngine

class EmbeddingEngine { // Initialize the model (downloads on first use, ~90MB) async initialize(): Promise<void> // Embed a single text string → 768-dimensional vector async embed( text: string, prefix?: string ): Promise<Float32Array> // Embed with query rewriting for matching (handles need/offer asymmetry) async embedForMatching( text: string, itemType: "need" | "offer", prefix?: string ): Promise<Float32Array> // Embed multiple texts in a batch async embedBatch( texts: string[], prefix?: string ): Promise<Float32Array[]> }

rewriteForMatching()

Rewrite a need/offer text to improve complementary matching. Strips demand framing from needs so they embed closer to matching offers.

function rewriteForMatching( text: string, itemType: "need" | "offer" ): string

perturbation.ts

Perturbation

Differential privacy via calibrated Laplace noise. Adds noise to embedding vectors before they leave the personal node.

perturbVector()

Perturb a vector with Laplace noise at a given epsilon value.

function perturbVector( vector: Float32Array, epsilon: number ): PerturbationResult

perturbWithLevel()

Perturb using named privacy levels instead of raw epsilon values.

function perturbWithLevel( vector: Float32Array, level: "low" | "medium" | "high" ): PerturbationResult

calibrateScale()

Calculate the Laplace scale parameter for a given epsilon and dimensionality.

function calibrateScale( epsilon: number, dimensions?: number // default: 768 ): number

sampleLaplace()

Sample a single value from a Laplace distribution.

function sampleLaplace(scale: number): number
Privacy Levels

Named levels map to epsilon values: low = 5.0 (sim ~0.994, minimal privacy), medium = 1.0 (sim ~0.880, recommended default), high = 0.1 (sim ~0.179, strong privacy but low utility).

vectors.ts

Vectors

Vector math utilities optimized for Float32Array. Used internally by embedding, perturbation, and matching.

cosineSimilarity()

function cosineSimilarity( a: Float32Array, b: Float32Array ): number // Returns value between -1 and 1

dotProduct()

function dotProduct( a: Float32Array, b: Float32Array ): number

l2Norm()

function l2Norm(v: Float32Array): number

normalize()

function normalize(v: Float32Array): Float32Array
Performance

All vector functions operate on Float32Array for memory efficiency and cache-friendly access. A 768-dim vector takes 3KB as Float32Array vs 6KB as a regular number array.
